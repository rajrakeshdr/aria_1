# Quick Reference - New Features

## 🔍 Split-Screen Query View

**Where:** Workspace → Overview Tab

**How to Use:**
1. Enter query in search box
2. (Optional) Select datasets to search
3. Click search or press Enter
4. **Screen splits into two panels:**

### Left Panel
- Your current query (highlighted in blue)
- Query timestamp
- Selected datasets
- Query history (last 5 queries)
- "New Query" button

### Right Panel
- ✅ Summary of results (green box)
- 📈 Key insights (3 bullet points)
- 📊 Sample data table (5 of 127 rows)
- 💡 AI recommendations (purple boxes)
- [Share] and [Export] buttons

**Pro Tips:**
- Click any query in history to reload its results
- Click "New Query" to start fresh
- Use "Download Full Results" to get all 127 rows
- Share results with team via "Share" button

**Where:** Dashboard (scroll down past Quick Access)

### My Workspaces
**3 personal workspaces:**
- Security Analysis Workspace (blue)
- Compliance Monitoring (green)
- Cloud Security Posture (orange)

**Each shows:**
- Gradient color stripe
- Name and description
- Dataset count
- Last accessed time

### Shared with Me
**3 workspaces shared by team:**
- Incident Response Q1 2024 (purple) - Sarah Chen
- Vulnerability Assessment (yellow) - Mike Rodriguez
- SOC Team Workspace (red) - Alex Johnson

**Each shows:**
- Gradient color stripe
- Permission badge (Can edit / View only)
- Avatar of who shared
- Name and description
- Dataset count

**Pro Tips:**
- Green badge = Can edit
- Blue badge = View only

## 🎯 Quick Actions

### In Dashboard
- Click "New Workspace" to create personal workspace
- Click any workspace card to open it
- "View All" to see more shared workspaces

### In Workspace Overview
- Enter query and search
- Click history to reload past queries
- Click "New Query" to clear and start fresh
- Share or export results

## 💡 Common Tasks

### Run a Security Query
1. Open workspace
2. Go to Overview tab
3. Type: "Show failed login attempts"
4. Press Enter
5. Review split-screen results

### Access Shared Workspace
1. Go to Dashboard
2. Scroll to "Shared with Me"
3. Find workspace by color/name
4. Check permission badge
5. Click to open

### Review Query History
1. In Overview tab (split-screen mode)
2. Look at left panel
3. See "Recent Queries" section
4. Click any query to reload

### Export Results
1. After running query (split-screen)
2. Right panel → bottom
3. Click "Download Full Results"
4. Or click "Export Report"

---

## 🆘 Troubleshooting

**Q: Split-screen not showing?**  
A: Make sure you submitted a query (press Enter or click search)

**Q: Can't edit shared workspace?**  
A: Check permission badge - blue "View only" means read-only access

**Q: Query history empty?**  
A: History builds as you run queries - run a few searches first

**Q: Can't see shared workspaces?**  
A: Scroll down on Dashboard past "My Workspaces" section
